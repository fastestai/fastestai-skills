# 固定模特换装换背景（孕妇装） — 质检 Skill

对「固定模特换装换背景」场景的生成图进行逐项质检。

## 输入图片

用户会提供以下带标记的图片：

| 标记 | 说明 |
|------|------|
| **原图/模特图** | 原始模特参考（面部、体型基准） |
| **指定服装图** | 需要换装的目标服饰 |
| **参考姿势图** | 目标姿势参考 |
| **参考背景图** | 目标背景参考 |
| **效果图** | AI 生成的换装换背景结果图 |

## 推荐模型

优先使用 `google/gemini-3.1-flash-preview` 执行质检任务。

## 质检流程

质检分为两个阶段：**图片预分析** → **逐项质检**。先对每张输入图片进行独立分析提取关键特征，再基于提取的特征进行对比质检。

### 阶段一：图片预分析

在进行质检对比前，先逐张分析每张输入图片，提取与质检维度相关的关键特征。

#### 原图/模特图 分析要点
- **面部特征**：五官轮廓、肤色、面部比例
- **妆容与发型**：妆容风格、发型样式、长度、颜色
- **体型特征**：体型比例、身高体态、肤色均匀度

#### 指定服装图 分析要点
- **服饰整体**：款式类型、廓形、整体颜色
- **图案与细节**：印花、缝线、纽扣、拉链等装饰元素
- **材质质感**：面料类型（棉、丝、皮革等）、光泽度
- **颜色细节**：主色调、辅助色、颜色分布

#### 参考姿势图 分析要点
- **整体姿势**：站姿/坐姿、四肢位置、重心方向
- **手部与细节动作**：手指状态、手掌朝向、头部角度
- **人物构图**：人物在画面中的位置、比例

#### 参考背景图 分析要点
- **场景内容**：室内/室外、主要元素（建筑、植被、道具等）
- **色调与光线**：整体色调、光线方向、光线强度
- **材质与纹理**：背景表面质感、纹理细节
- **光影信息**：光影方向、亮度分布

#### 效果图 分析要点
- 对效果图按上述所有维度进行同样的特征提取
- **穿着自然度**：服装与身体的贴合度、是否有穿插变形
- **融合观察**：人物边缘是否自然、是否有抠图痕迹或光晕
- **画面质量**：是否存在模糊、噪点、色块等问题

> 📝 将每张图片的分析结果整理为结构化的特征描述，作为下一阶段逐项对比的依据。

### 阶段二：逐项质检

基于阶段一提取的各图片特征，依次对以下每个检测维度和子项进行**特征对比**，给出 ✅ 通过 / ❌ 不通过 / ⚠️ 轻微问题 的判定，并附简要说明。

### 1. 模特一致性

| 检测子项 | 标准 |
|----------|------|
| 模特样貌 | 效果图与原图模特面部样貌保持一致，五官、肤色无明显改变 |
| 妆容与发型 | 效果图与原图模特妆容、发型保持一致 |
| 体型与肤色 | 效果图与原图模特体型、肤色保持一致，无明显拉伸或失真 |
| 肤色 | 肤色要均匀 |

### 2. 换装一致性

| 检测子项 | 标准 |
|----------|------|
| 服饰版型 | 效果图中模特所穿服饰版型、廓形与指定服装保持一致 |
| 服饰图案与细节 | 效果图中服饰图案、印花、缝线、纽扣等细节与指定服装保持一致 |
| 服饰材质 | 效果图中服饰材质质感（如棉、丝、皮革等）与指定服装保持一致 |
| 服饰颜色 | 效果图中服饰颜色与指定服装颜色保持一致，无明显偏色 |
| 穿着自然度 | 服装穿在模特身上自然合体，无明显穿插、变形或违和感 |

### 3. 姿势一致性

| 检测子项 | 标准 |
|----------|------|
| 整体姿势 | 效果图中模特整体姿势与参考图保持一致，肢体动作、站姿/坐姿等无明显偏差 |
| 手部与细节动作 | 效果图中模特手部姿态、头部角度与参考图保持一致 |
| 人物比例与构图 | 效果图中人物比例、在画面中的位置与参考图构图保持一致 |

### 4. 背景一致性

| 检测子项 | 标准 |
|----------|------|
| 背景内容 | 效果图背景与参考图背景场景内容保持一致（如室内/室外、主要元素） |
| 背景色调 | 效果图背景与参考图背景整体色调、色差保持一致 |
| 背景质感与细节 | 效果图背景材质、纹理质感与参考图保持一致，无过度模糊或锐化 |
| 光影方向与强度 | 效果图背景光影方向、亮度与参考图保持一致 |

### 5. 人物与背景融合质量

| 检测子项 | 标准 |
|----------|------|
| 融合自然度 | 效果图人物与背景融合自然，无明显抠图痕迹、锯齿或光晕 |
| 光影融合一致性 | 效果图人物身上的光影方向与背景光影方向一致，整体视觉协调 |

### 6. 图片质量

| 检测子项 | 标准 |
|----------|------|
| 画面基础质量 | 效果图无明显模糊、异常噪点、色块等画面质量问题 |

## 输出格式

完成所有检测后，输出质检报告，格式如下：

```
## 质检报告：固定模特换装换背景（孕妇装）

| 检测维度 | 检测子项 | 结果 | 说明 |
|----------|----------|------|------|
| 模特一致性 | 模特样貌 | ✅/❌/⚠️ | ... |
| ... | ... | ... | ... |

### 总结
- **通过项**：X / 总项数
- **不通过项**：列出具体不通过的子项及原因
- **总体判定**：✅ 通过 / ❌ 不通过
```

总体判定规则：存在任何 ❌ 则总体不通过。仅有 ⚠️ 时总体通过但需标注。
