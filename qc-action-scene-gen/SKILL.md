# 固定模特&背景，根据参考动作生成场景图 — 质检 Skill

对「根据参考动作生成场景图」场景的生成图进行逐项质检。

> ⚠️ **硬性约束：生成的场景图不能带有任何文字**

## 输入图片

用户会提供以下带标记的图片：

| 标记 | 说明 |
|------|------|
| **原图/模特图** | 原始模特参考（面部、身材基准） |
| **参考动作图** | 目标动作/姿势参考 |
| **参考服饰图** | 目标服饰参考 |
| **参考背景图** | 目标背景参考 |
| **效果图** | AI 生成的场景结果图 |

## 质检流程

依次对以下每个检测维度和子项进行检查，给出 ✅ 通过 / ❌ 不通过 / ⚠️ 轻微问题 的判定，并附简要说明。

### 1. 模特一致性

| 检测子项 | 标准 |
|----------|------|
| 模特样貌与身材 | 效果图与原图模特面部样貌、身材比例保持一致 |
| 模特发型 | 效果图与原图模特发型保持一致，无明显变化 |
| 模特姿势 | 效果图人物姿势与参考动作图保持一致 |
| 肢体完整性 | 效果图模特手、脚、手指等肢体比例正常，无一大一小、多指、关节扭曲等畸形问题 |

### 2. 服饰一致性

| 检测子项 | 标准 |
|----------|------|
| 服饰整体 | 效果图与参考服饰整体款式、颜色保持一致 |
| 服饰细节与图案 | 效果图与参考服饰的图案、印花、缝线等细节保持一致 |
| 服饰材质 | 效果图与参考服饰材质质感保持一致，无明显失真 |
| 服饰 Logo | 效果图中服饰 Logo 与参考服饰保持一致，清晰可辨，无变形或缺失 |

### 3. 背景一致性

| 检测子项 | 标准 |
|----------|------|
| 背景内容 | 效果图背景与参考背景图场景内容保持一致 |
| 背景色调 | 效果图背景与参考图背景色调、色差保持一致 |

### 4. 人物与背景融合质量

| 检测子项 | 标准 |
|----------|------|
| 融合自然度 | 效果图人物与背景融合自然，无明显浮空感或抠图痕迹 |

### 5. 画面整洁度

| 检测子项 | 标准 |
|----------|------|
| 无多余文字 | 效果图画面中不得出现任何文字（含水印、字幕、标注等） |

> 🚨 **这是硬性不通过项**：只要出现任何文字即判定该项 ❌ 不通过。

### 6. 光线处理

| 检测子项 | 标准 |
|----------|------|
| 光线自然度 | 效果图中人物光线方向、强度与背景场景光线保持一致，户外场景尤其需保证光影合理自然 |

### 7. 图片质量

| 检测子项 | 标准 |
|----------|------|
| 画面基础质量 | 效果图无明显模糊、异常噪点、色块等画面质量问题 |

## 输出格式

完成所有检测后，输出质检报告，格式如下：

```
## 质检报告：固定模特&背景 · 参考动作生成场景图

| 检测维度 | 检测子项 | 结果 | 说明 |
|----------|----------|------|------|
| 模特一致性 | 模特样貌与身材 | ✅/❌/⚠️ | ... |
| ... | ... | ... | ... |

### 总结
- **通过项**：X / 总项数
- **不通过项**：列出具体不通过的子项及原因
- **总体判定**：✅ 通过 / ❌ 不通过
```

总体判定规则：存在任何 ❌ 则总体不通过（尤其注意「无多余文字」为硬性项）。仅有 ⚠️ 时总体通过但需标注。
