# 一键换背景 — 质检 Skill

对「一键换背景」场景的生成图进行逐项质检。

## 输入图片

用户会提供以下带标记的图片（至少包含原图和效果图）：

| 标记 | 说明 |
|------|------|
| **原图** | 原始模特图（含服饰、配饰、阴影等） |
| **参考背景图** | 目标背景参考 |
| **效果图** | AI 生成的换背景结果图 |

## 推荐模型

优先使用 `google/gemini-3.1-flash-preview` 执行质检任务。

## 质检流程

质检分为两个阶段：**图片预分析** → **逐项质检**。先对每张输入图片进行独立分析提取关键特征，再基于提取的特征进行对比质检。

### 阶段一：图片预分析

在进行质检对比前，先逐张分析每张输入图片，提取与质检维度相关的关键特征。

#### 原图 分析要点
- **模特面部**：五官轮廓、肤色、面部比例
- **妆容与发型**：妆容风格、发型样式、长度、颜色
- **姿势与肢体**：整体姿势、四肢位置、手部状态
- **服饰特征**：款式类型、颜色、图案、印花、缝线等细节、材质质感
- **配饰与随身物品**：项链、手链、耳环、发带、美甲、手持包包等
- **阴影信息**：人物是否有阴影、阴影方向与强度

#### 参考背景图 分析要点
- **场景内容**：室内/室外、主要元素（建筑、植被、道具等）
- **色调与光线**：整体色调、光线方向、光线强度
- **阴影信息**：场景中是否有阴影
- **人物布局参考**：如背景图中有人物，记录其比例与位置

#### 效果图 分析要点
- 对效果图按上述所有维度进行同样的特征提取
- **融合观察**：人物边缘是否自然、是否有抠图痕迹
- **画面质量**：是否存在模糊、噪点、色块等问题

> 📝 将每张图片的分析结果整理为结构化的特征描述，作为下一阶段逐项对比的依据。

### 阶段二：逐项质检

基于阶段一提取的各图片特征，依次对以下每个检测维度和子项进行**特征对比**，给出 ✅ 通过 / ❌ 不通过 / ⚠️ 轻微问题 的判定，并附简要说明。

### 1. 模特一致性

| 检测子项 | 标准 |
|----------|------|
| 模特样貌 | 效果图与原图模特面部样貌保持一致，肤色无明显改变 |
| 妆容与发型 | 效果图与原图模特妆容、发型保持一致 |
| 姿势 | 效果图与原图模特姿势、肢体动作保持一致 |

### 2. 服饰一致性

| 检测子项 | 标准 |
|----------|------|
| 服饰版型 | 效果图与原图模特所着服饰版型、颜色保持一致 |
| 服饰图案与细节 | 效果图与原图服饰图案、印花、缝线等细节保持一致 |
| 服饰材质 | 效果图与原图服饰材质质感（如棉、丝、皮革等）保持一致 |

### 3. 配饰与随身物品一致性

| 检测子项 | 标准 |
|----------|------|
| 穿戴配饰 | 效果图与原图模特所戴配饰（项链、手链、耳环、发带、美甲等）保持一致 |
| 手持包包 | 效果图与原图模特手持包款、颜色、细节保持一致 |

### 4. 背景一致性

| 检测子项 | 标准 |
|----------|------|
| 背景内容 | 效果图背景与参考图背景场景内容保持一致 |
| 背景色调 | 效果图背景与参考图背景色调、色差保持一致 |

### 5. 人物与背景融合质量

| 检测子项 | 标准 |
|----------|------|
| 融合自然度 | 效果图人物与背景融合自然，无明显浮空感或抠图痕迹 |
| 人物比例与布局 | 效果图人物比例、在画面中的位置与参考背景图中人物比例、布局保持一致 |

### 6. 阴影处理

根据实际情况选择适用的规则进行检查：

| 检测子项 | 标准 |
|----------|------|
| 原图有阴影 → 效果图消除阴影 | 原图人物有阴影时，效果图中阴影需完全消除 |
| 原图有阴影 → 效果图保留阴影 | 原图人物有阴影时，效果图中阴影需自然保留 |
| 参考图有阴影 → 效果图消除阴影 | 参考背景图中有阴影时，效果图中该阴影需消除 |
| 原图与参考图均有阴影 → 以原图为准 | 两者均有阴影时，效果图阴影呈现以原图人物阴影为参考基准 |

> 注意：阴影规则需根据运营需求选择适用项，不一定全部适用。

### 7. 图片质量

| 检测子项 | 标准 |
|----------|------|
| 画面基础质量 | 效果图无明显模糊、异常噪点、色块等画面质量问题 |

## 输出格式

完成所有检测后，输出质检报告，格式如下：

```
## 质检报告：一键换背景

| 检测维度 | 检测子项 | 结果 | 说明 |
|----------|----------|------|------|
| 模特一致性 | 模特样貌 | ✅/❌/⚠️ | ... |
| ... | ... | ... | ... |

### 总结
- **通过项**：X / 总项数
- **不通过项**：列出具体不通过的子项及原因
- **总体判定**：✅ 通过 / ❌ 不通过
```

总体判定规则：存在任何 ❌ 则总体不通过。仅有 ⚠️ 时总体通过但需标注。
